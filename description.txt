Ensure your mods work seamlessly together with Compatibility Checker!

This utility mod helps players and developers identify potential incompatibilities between mods, saving time and frustration before starting a playthrough. Powered by a community-driven online database, it provides live reports on the compatibility of your enabled mods and offers suggestions for resolving conflicts.

Features:

• Online Compatibility Database
  - Access a comprehensive database of compatibility information for dozens of mods, updated in real-time by the community.

• Multiplayer Compatibility Checks
  - Verify that your mod setup works well in multiplayer environments.

• Recommended Fixes
  - Get suggestions for mods to install that can resolve or mitigate compatibility issues.

• Enabled Mods Overview
  - Toggle the option to display a list of your enabled mods directly on the main menu.

• GitHub Issue Integration
  - Displays relevant GitHub issues for detected compatibility problems, perfect for developers and technical users.

NOTE: This mod is in an early Beta stage. Bugs and incomplete features may exist.
